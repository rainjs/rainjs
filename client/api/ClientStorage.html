Class ClientStorage¶

Creates a new client storage for the current context.

It provides methods to work with the client data. The storage methods can be used directly from the the controller’s context through the storage key.

Examples¶

this.context.storage.set('cart', { items: [1, 2, 3, 4], total: 10});
var data = this.context.storage.get('cart');
console.log(data);

// That will print
//     {
//         items: [1, 2, 3, 4],
//         total: 10
//     }
