Class SyslogAppender¶

The syslog appender class is used for redirecting the log output to the syslog service:

{

“level”: “debug”, “type”: “syslog”, “layout”: {

“type”: “pattern”, “params”: {

“pattern”: “[%level] %date %logger: %message%newline%stacktrace”

}

}, “params”: {

“identity”: “”, “facility”: “LOCAL0”

}

}

Constructor¶

class SyslogAppender(level, layout, options)¶
Arguments:
  • level (String) – the log level for this instance
  • layout (Layout) – the log layout to use
  • options (Object) – the configuration options
  • options.file (String) – the file to write to
  • options.identity (String) – a string to be added to every log message (default is “”)
  • options.facility (String) – the syslog facility to log into (default is “LOCAL0”)
