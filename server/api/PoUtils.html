parsePo¶

PoUtils.parsePo(data)¶
Arguments:
  • data (String) – the po data to be parsed
Returns Object:

the object containing the parsed po data

Parses the po data and creates an object using it. This object has the following structure: {

“”: {key1: value1, ..., keyN: valueN}, //headers “msgid1”: [null, “translation”], “msgid2”: [“msgidPlural”, “singular”, “pluralForm0”, “pluralForm1”, ...], ... “msgidN”: [null, “translation”]

}
