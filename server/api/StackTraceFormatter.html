Class StackTraceFormatter¶

Cross-browser stack trace formatter. Supports Firefox, Chrome, IE and Safari as well as node.js.

Note: it currently does not aim to reproduce the same stack trace for every browser.

class StackTraceFormatter(error)¶

Methods¶

format¶

StackTraceFormatter#format()¶
Returns String:the formatted stack trace

Formats the stack trace for a meaningful inspection. Removes the inner functions from RAIN that appear at the top of the stack and ensures cross-browser functionality (e.g. IE support).
