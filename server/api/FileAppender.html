Class FileAppender¶

The file appender class is used for redirecting the log output to a file:

{
    "level": "debug",
    "type": "file",
    "layout": {
        "type": "pattern",
        "params": {
            "pattern": "[%level] %date %logger: %message%newline%stacktrace"
        }
     },
     "params": {
         "file": "/path/to/file",
         "encoding": "utf-8",
         "mode": "0644"
     }
}

Constructor¶

class FileAppender(level, layout, options)¶
Arguments:
  • level (String) – the log level for this instance
  • layout (Layout) – the log layout to use
  • options (Object) – the configuration options
  • options.file (String) – the file to write to
  • options.encoding (String) – the encoding to use for the log file (Default: ‘utf-8’)
  • options.mode (String) – the permissions to use when creating a new log file (Default: ‘0644’)
