getLatestVersion¶

ComponentRegistry#getLatestVersion(componentId[, fragment])¶
Arguments:
  • componentId (String) – the id of the component
  • fragment (String) – a version fragment
Returns String:

the latest version of the component of undefined if it isn’t found

Returns the latest version for a component. This method also supports specifying version fragments in the second optional parameter. This means that if the provided version is “1”, it will return the latest version that has “1” as its major version (like “1.8.5”). You can also specify an exact version in the fragment. If the component isn’t found, it returns undefined. Also, if you provide a fragment that it is too big, it will return undefined (like 3.2 and the latest version is 2.5.3).
